## üöÄ Getting Started

1. Install dependencies:
   
```
bash
   npm install
   
```

2. Set up your `.env` file in the `backend` folder:
   
```
   PORT=
   DB_NAME=
   DB_USER=
   DB_PASSWORD=
   JWT_SECRET=
   GENAI_API_KEY=
   CASHFREE_APP_ID=
   CASHFREE_SECRET_KEY=
   
```

3. Start the server:
   
```
bash   node backend/server.js
   
```

4. Open your browser and go to:
   
```
   http://localhost:3000
   
```
